## 3. 交互设计

### 3.1 用户操作流程

```
[页面加载] → [展示列表] → [用户点击新增]
  │
  ▼
[弹出表单] → [填写信息] → [提交]
  │                           │
  │                     ┌─────┴─────┐
  │                     │           │
  │                  [成功]      [失败]
  │                     │           │
  │              [关闭弹窗]   [显示错误]
  │              [刷新列表]   [保持弹窗]
```

### 3.2 交互细节

| 操作 | 触发 | 响应 | 备注 |
|------|------|------|------|
| 点击新增 | click 新增按钮 | 弹出表单弹窗 | |
| 提交表单 | click 提交按钮 | loading → 成功/失败 | 防重复提交 |
| 删除 | click 删除按钮 | 确认弹窗 → 删除 | 二次确认 |
| 搜索 | input + debounce | 请求搜索接口 | 防抖 |

### 3.3 加载与空状态

| 状态 | 展示 |
|------|------|
| 首次加载 | 骨架屏 / Loading 指示器 |
| 加载更多 | 底部 loading |
| 空数据 | 空状态插图 + 引导文案 |
| 加载失败 | 错误提示 + 重试按钮 |
